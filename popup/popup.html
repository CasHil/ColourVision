ColourVision

By Casper Hildebrand

Our goal at ColourVision is to make the web more accessible for colourblind people in a transparent way.

Before completing the installation of this plugin, you were asked if you were okay with the plugin being able to read and change all your data on all sites. The plugin needs this access to be able to change the colours on the web page and nothing else.

Many browser plugins are not open-source, meaning you have to trust that it only does what it tells you it is doing. We believe that transparency is important when dealing with information that is as sensitive as your browsing behaviour. Therefore, anyone may inspect the code by clicking the image in the top right and suggest changes. You can also see what data the plugin is collecting by clicking the link in the bottom right.
